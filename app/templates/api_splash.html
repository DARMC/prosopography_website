PROSOPOGRAPHY OF TRAVELERS DATA API


The Digital Atlas of Roman and Medieval Civilizations is proud to provide a an API for for the Prosopography of Mediterranean Travelers that will allow you to integrate the data into your own projects freely. As it is developed, the API will allow queries by traveler, date, and place and return JSON objects from the current state of the database. The API currently gives responses to two types of URI queries.

../api/v1/places/[place name]
Returns a list of JSON objects for [place name], where place name is any string corresponding to a place in the travelers database (case insensitive). For place names consisting of multiple words, words should be separated by an underscore.
../api/v1/traveler/[traveler name]

Returns a list of JSON objects for [traveler name], where traveler name is any string corresponding to an individual in the traelers database (case insensitive). For each traveler matching [traveler name], an ordered list of JSON objects is returned, containing that traveler's known movements. For names consisting of multiple words, words should be separated by an underscore.

Note: DARMC expressly prohibits the use of this API feature to create a complete copy of the travelers table. Please contact us personally if you would like to use a complete copy of this data in your research.
